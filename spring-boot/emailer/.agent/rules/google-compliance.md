1 Google Java Style Guide

Follow Google Java Style Guide strictly
Use 2 spaces for indentation (no tabs)
Maintain 100 character line length limit
Do not use wildcard imports
Use K&R style braces (opening brace on same line)

2 Javadoc Requirements

Javadoc is required for all public classes
Javadoc is required for all public methods
Follow Google Javadoc style
First sentence must be a summary
Use @param, @return, @throws tags appropriately
Document behavior, not generated code
